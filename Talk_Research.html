Implementation

Implementation

  • Implemented interfaces:
    • Merlin (Dense/Frame based format) data and live interface
    • AdvaCam TimePix3 (Sparse/Event based format) data interface
    • ASI TimePix3 (Sparse/Event based format) data and live interface in development
  • Implemented as GUI application and CLI tool fully in C++

  • Performance:
    • Frame based format: ~30 kHz
    • Event based format: >100 kHz

Summary

  • Very fast algorithm with tunable contrast transfer, well suited for live-imaging
  • Accurate detection, localisation of atoms possible at very low doses
  • Resolution limited by scan step size
  • No reliable thickness information
